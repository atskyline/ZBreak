## 1.0.1
 - 将基本工作时间由2个小时调整为1个小时
 - 引入最小休息时间概念，每次休息至少TimerInterval × MinBreakCount 毫秒（为了排除掉去上个厕所就被认为是休息的情况）
 - 在托盘提示中显示已经连续工作的时间
 - 修复了一个由于锁屏状态会崩溃的bug。
 - 修复了连续工作时间显示的BUG

# Other
- 有一个逻辑上的处理，提示窗显示的时候也是计时的。所以如果开着提示框超过10分钟，然后点击“我要再干十分钟”，会导致1分钟左右后，又给出提醒。（1分钟是活动判定间隔）
